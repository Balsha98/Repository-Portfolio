# BAZOVICH.DEV - Frontend Engineer Portfolio

My personal portfolio website showcasing my projects, skills, and experience as a frontend engineer. Built with modern web technologies and deployed on a custom domain.

## Features

- **Modern Design** - Clean and professional user interface.
- **Responsive Layout** - Optimized for all devices & screen sizes.
- **Fast Performance** - Built with vite for optimal loading speeds.
- **Project Showcase** - Interactive display of development projects.
- **About Section** - Skills and professional background.
- **Contact Information** - Easy ways to get in touch.
- **Smooth Animations** - Engaging user experience with CSS3 transitions.
- **Custom Domain** - Professional web presence @ bazovich.dev.

## Installation

### Prerequisites

- Node.js 16.0 or higher.
- npm (Node Package Manager).

### Setup Instructions

1. Clone the repository:

```bash
git clone https://github.com/Balsha98/Repository-Portfolio.git
```

2. Navigate to the project directory:

```bash
cd portfolio
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

5. Open your browser and visit:

```
http://localhost:5173
```

## Building for Production

Create an optimized production build:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Deployment

This portfolio is deployed on Vercel with a custom domain:

### Deploy to Vercel

1. Push your code to GitHub.
2. Import your repository in Vercel.
3. Configure build settings:
    - Build Command: `npm run build`
    - Output Directory: `dist`
4. Add your custom domain in Vercel's domain settings.
5. Deploy!

### Custom Domain Setup

- Domain: `bazovich.dev`
- DNS configured through Vercel.
- Automatic HTTPS with SSL certificate.
